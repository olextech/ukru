## Installation

1. Copy files uk-RU.keylayout and uk-RU.icns to a system folder /Library/Keyboard Layouts/
2. Open System preferences > Keyboard > Input sources, press +, scroll to "Others" and pick source "Russian"
3. Restart your Mac
